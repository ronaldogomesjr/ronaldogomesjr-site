PATCH — contato bilíngue corrigido + remoção efetiva de Pesquisa/Research

Arquivos incluídos neste patch:
- assets/admin.js
- assets/cms-render.js
- assets/footer-links.js
- assets/site-content.js
- admin/config.yml
- admin/index.html
- vercel.json
- páginas HTML em /pt e /en com o menu estático atualizado e cache-busting nos scripts

O patch NÃO inclui arquivos content/*.json, para não sobrescrever conteúdo já atualizado no painel.

O que este patch corrige:
1. A página /en/contact/ passa a usar descricao_en quando existir.
2. Os contatos antigos que ainda usam o campo legado tipo recebem fallback correto em inglês:
   - Perfil acadêmico → Academic profile
   - Currículo → CV
   - Contato → Contact
3. O painel continua permitindo editar:
   - descrição em português
   - description in English
4. A chave de agrupamento dos contatos passa a ser o nome do contato, não nome + ordem. Isso evita que duplicatas antigas PT/EN fiquem desencontradas.
5. Pesquisa/Research é removida do menu estático e do menu carregado por JavaScript.
6. /pt/pesquisa/ e /en/research/ passam a redirecionar pela Vercel para /pt/projetos/ e /en/projects/.

Importante sobre excluir a página Pesquisa/Research:
Subir arquivos pelo GitHub substitui ou cria arquivos, mas não apaga arquivos antigos automaticamente.
Este patch faz o site redirecionar essas URLs. Para apagar do repositório também, delete manualmente no GitHub:
- pt/pesquisa/index.html
- en/research/index.html

Depois de aplicar:
1. Aguarde a Vercel publicar.
2. Abra em aba anônima.
3. Teste /en/contact/.
4. Teste /en/research/; ela deve redirecionar para /en/projects/.
5. Teste /pt/pesquisa/; ela deve redirecionar para /pt/projetos/.
