# Ronaldo Gomes Jr. Site — V19 Static

Versão estática, sem Next.js, sem npm install e sem build.

Como publicar:
1. Suba todos os arquivos desta pasta no GitHub.
2. Na Vercel, importe o repositório como projeto estático.
3. Não use Build Command.
4. Não use Output Directory.
5. O site deve publicar diretamente.

Estrutura:
- /pt/
- /en/
- /assets/style.css
- /assets/site.js
